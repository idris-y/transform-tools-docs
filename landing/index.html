Enhance your Blender workflow with custom 3D gizmos. Define start and end states to execute complex pose-to-pose adjustments and constrained alignments in seconds.

The Dual-Gizmo
Workflow

Transform Tools is built to give you exact mathematical control over how your elements move, rotate, and scale. By acting as a seamless data bridge, it allows you to effortlessly push and pull precise orientations between your Objects, custom Gizmos, and Blender's 3D Cursor.

01 / Define Previous

Initial
State

Set the initial state by clicking, using the 3D cursor, or automatically generating a gizmo from your selected mesh data.

02 / Define Active

Target
Destination

Set the target destination and orientation. Transform Tools calculates the exact spatial difference between the two gizmos.

03 / Execute

Perfect
Alignment

Instantly transform, move, rotate, or perfectly align your selection from the Previous state to the Active state.
